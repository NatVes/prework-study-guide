HTML

  • The head element contains information (Metadata) about the webpage
  • !DOCTYPE html - declaration, not case-sensitive
  • HTML elements - case-sensitive
  • Child element - element that is nested inside other element
  • Parent elemen - elemen that holds all the child elements (example - body element)
  • The body element represents the visible content shown to the user
  • h1...h6 - levels of heading

CSS

  • three ways to style a webpage using CSS: inline CSS, internal CSS, and external CSS
  • CSS Rule:
    • selector defines the element or attributes to which the rules, or declarations, will apply
    • declarations contain the CSS property we want to apply and the value of the property
  • a margin property indicates how much space we want around the outside of an element
  • a padding property adds space around the content inside an element

Git

  • Git is a version control system
  • GitHub is a cloud-based hosting service
  • Steps:
    • create a feature branch
    • add modifications to the staging area
    • sync with the base branch in GitHub
    • push changes to the remote branch
    • "Compare & pull" request in the GitHub repo
    • give the pull request (PR) a descriptive title
    • "Create Pull Request"
    • "Merge Pull Request"
  • git clone: retrieve an entire repository from a hosted location via URL
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: add all changes to your next commit (stage)
  • git commit -m “[descriptive message]”: commit your staged content as a new commit snapshot
  • git pull: fetch and merge any commits from the tracking remote branch
  • git push [alias] [branch]: transmit local branch commits to the remote repository branch

JavaScript

  • JS - high-level dynamic programming language with dynamic types
  • JS is case-sensitive
  • Variables - containers that store values
  • Array - a single variable that is used to hold a group of data
  • A single array can store different types of data
  • Functions - way of packaging functionality that you wish to reuse
  • Control flow - order in which a comp. executes code in a script
  • Conditional logic: IF statement
  • FOR loop:
    • starting point var x=0;
    • condition x<'varname.length;
    • itration x++
